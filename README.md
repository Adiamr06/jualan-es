# Tea Fresh Web Application

A premium, responsive landing page for a modern tea shop brand.

## Features
- **5 Variants Showcase**: Jasmine Green Tea, Thai Milk Tea, Peach Blossom, Lychee Rose, and Taro Creamy.
- **Dynamic Shopping Cart**: Interactive cart with real-time updates.
- **Modern UI/UX**: Glassmorphism effects, smooth animations, and premium aesthetics.
- **SEO Optimized**: Semantic HTML and meta tags for search engine visibility.
- **Professional Structure**: Organized folder system for scalability.

## Tech Stack
- HTML5
- CSS3 (Vanilla + Custom Properties)
- JavaScript (Vanilla ES6)

## Folder Structure
```text
/
├── index.html          # Main HTML entry
├── css/
│   ├── variables.css   # Color palette & variables
│   ├── components.css  # Reusable UI elements
│   └── main.css        # Page layout & animations
├── js/
│   ├── main.js         # Product data & UI logic
│   └── cart.js         # Cart state management
└── assets/
    └── img/            # High-resolution product images
```

## How to Run / Hosting
### Local (XAMPP/WAMP)
1. Place the `Aplikasi` folder inside your `htdocs` directory.
2. Start Apache via XAMPP Control Panel.
3. Open `http://localhost/Aplikasi` in your browser.

### Online Hosting (Netlify/Vercel/GitHub Pages)
1. Upload these files to a GitHub repository.
2. Connect the repository to your hosting provider.
3. The application will be automatically deployed as a static site.
